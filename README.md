## Air Typing (version 1.1.0)

### Feature
**支持手势：**
1. 整个手掌按压完成一次按键，输入一个字符
2. 右手左滑删除末尾字符
3. 左手左滑删除整个正在输入或已经输入的单词
4. 左手握拳选择下一个候选词
5. 右手握拳确认正在输入的单词

**算法：**
每个单词的第一次字符根据按压位置进行概率计算，之后的字符根据和前一个字符的相对位置进行概率计
算。

计算概率时考虑了词频。

坐标系以键盘左上角作为原点，按键大小作为单位长度。
